# ⚙️ Setup Guide

1. Open Roblox Studio
2. Insert folders:

ReplicatedStorage
ServerScriptService
StarterPlayerScripts

3. Place scripts accordingly:

- Modules → ReplicatedStorage
- Server scripts → ServerScriptService
- Client scripts → StarterPlayerScripts

4. Enable API Services:
Home → Game Settings → Security → Enable Studio Access to API Services

5. Run the game and test currency updates, shop purchases, and trades.
