# 💰 Roblox Game Economy System

A complete, modular economy system built in Roblox using Lua.  
Designed for scalability, security, and real-world game usage.

---

## 🚀 Features

- Currency management (earn, spend, update)
- Shop system with validation
- Player-to-player trading
- Transaction logging system
- Anti-exploit validation
- Data persistence (DataStore integration)

## 🧠 System Overview

This system is structured using a modular architecture:

- **CurrencyManager** → Handles all currency operations  
- **DataManager** → Manages saving/loading player data  
- **TradeManager** → Handles secure player trades  
- **TransactionLogger** → Logs all transactions for debugging and auditing
